# TJOE Journal Site

Demo: http://tjoe.org.s3-website-us-east-1.amazonaws.com/

## Installation

```bash
npm install
npm start
open http://localhost:3500
```

## Building Static files

```bash
npm run build
```
Serve index.html, favicon.ico, and /static from server of your choice.
